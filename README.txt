# 🛒 E-Commerce UI Automation using Selenium WebDriver

This project automates key UI flows of an e-commerce website (`https://www.saucedemo.com/`) using **Selenium WebDriver**, **TestNG**, and the **Page Object Model** (POM) design pattern. It includes CI integration with Jenkins and test reporting with ExtentReports.

---

## ✅ Features Automated

- 🔐 Login functionality
- 🔍 Product search simulation
- 🛒 Add to Cart
- 💳 Checkout process
- 🌐 Cross-browser testing (Chrome & Firefox)
- 📸 Failure screenshot capture
- 📊 HTML test reporting (ExtentReports)

---

## 🧰 Tech Stack

| Tool               | Purpose                            |
|--------------------|------------------------------------|
| Selenium WebDriver | Browser automation                 |
| TestNG             | Test execution and reporting       |
| Maven              | Project build and dependency mgmt  |
| Jenkins            | CI/CD pipeline (auto test run)     |
| Git + GitHub       | Version control and collaboration  |
| ExtentReports      | HTML test reporting                |
