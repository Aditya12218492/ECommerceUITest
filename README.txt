## 🔍 Project Overview
This project automates an e-commerce website's core user interface workflows using Selenium WebDriver and Java.

## ✅ Features Automated
- Login
- Product Search
- Add to Cart
- Checkout

## 🧪 Tech Stack Used
- Java
- Selenium WebDriver
- TestNG
- Page Object Model (POM)
- Git & GitHub
- Jenkins (for CI)
- ChromeDriver & GeckoDriver

## 🖥️ How to Run
1. Clone the repo
2. Open in IntelliJ or Eclipse
3. Run the test cases from `TestRunner.java`
